Library

API for managing a library.

Books

API calls for the books object.

POST /books

Create a new book.

title string Title of the book.
author string Author of the book.
copies integer Number of copies of the book in the library.

Example

POST /books
{
  "title": "A Tale of Two Cities",
  "author": "Charles Dickens",
  "copies": 5
}

Responses

Success

The book was created successfully.

id integer ID of the book.
title string Title of the book.
author string Author of the book.
copies integer Number of copies of the book in the library.

Example

Response code 200
{
  "id": 1,
  "title": "A Tale of Two Cities",
  "author": "Charles Dickens",
  "copies": 5
}

Staff

API calls for managing staff members.

Employees

Staff members earning wages.

POST /employees

Create a new employee.

name string Name of the employee.
title string Job title of the employee.
favorite integer ID of their favorite book.

Example

POST /employees
{
  "name": "Bob Loblaw",
  "title": "Internet Marketing Manager",
  "favorite": 1
}

Volunteers

Staff members not earning wages.

POST /volunteers

Create a new volunteer.

name string Name of the volunteer.
title string Job title of the volunteer.

Example

POST /volunteers
{
  "name": "Vin Diesel",
  "title": "Bookmobile Driver"
}

PATCH /volunteers/<id>

Update the properties of a volunteer.

name string Name of the volunteer.
title string Job title of the volunteer.

Example

PATCH /volunteers/23
{
  "title": "Library Assistant"
}
